# rock-paper-scissor
python code to play rock, paper, and scissors games on an IDE. 

Contains two functions rps for selecting the winner and game for playing the game via different modes. 

the rules are as follows-
1- Rock vs Paper - Paper wins
2- Rock vs Scissors - Rock wins
3- Paper vs Scissors - Scissors wins


Rules for input - 
Provide R for Rock, P for paper, and S for scissors. They're Case sensitive.
